Judul Isi

Sederhananya, pemrograman adalah proses membuat program di komputer. Pembuatan program ini bisa berupa website, software, aplikasi android, dan sebagainya. Pemrograman dimulai melalui beberapa tahapan, seperti tulis-menulis, menguji, merevisi, mengevaluasi, serta pengujian kembali.

Bahasa pemrograman ini adalah satu set aturan sintaks dan semantik yang digunakan untuk mendefinisikan program komputer. contohnya: BASIC, C, C++, COBOL, FORTRAN, ada, Pascal, Java , .NET, Bahasa Assembly .
